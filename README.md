# MemberManagement

### 프로젝트 설명

   - 관리자의 회원관리를 위한 회원관리 App Sample입니다.

---------------------

#### 통합/회원 서비스
  - 로그인 / 회원가입 / 회원 정보수정 / 비밀번호 수정 / 프로필 사진 변경
#### 관리자 서비스
  - 회원목록 / 회원강퇴
